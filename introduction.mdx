## Overview

Artist Game empowers teams to create, organize, and maintain project documentation effortlessly. You get a centralized space to build structured docs with Markdown, MDX components, and version control integration. Whether you manage technical specs, user guides, or onboarding materials, Artist Game streamlines collaboration and keeps your knowledge base up-to-date.

<Callout kind="info">
Artist Game supports real-time editing, search across docs, and custom themes to match your brand.
</Callout>

## Quick Navigation Guide

Get up and running in minutes. Follow these steps to explore Artist Game:

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://app.artistgame.com` and start a new documentation project.
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    Click "New Page" and use the editor to write in Markdown or MDX.
  </Step>
  <Step title="Organize with Sidebar" icon="menu">
    Drag pages into collections and generate automatic navigation.
  </Step>
  <Step title="Publish and Share" icon="globe">
    Deploy your site and share the live URL with your team.
  </Step>
</Steps>

For a hands-on example, initialize a project using the CLI:

<CodeGroup tabs="npm,yarn">
  ```bash
  npm create artistgame@latest my-docs
  cd my-docs
  npm run dev
  ```
  ```bash
  yarn create artistgame my-docs
  cd my-docs
  yarn dev
  ```
</CodeGroup>
